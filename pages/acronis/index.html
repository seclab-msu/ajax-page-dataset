60440: macOS High Sierra(10.13) support in Acronis True Image

Also read in:

  • 日本語

use Google Translate

Applies to: 

  • Acronis Cyber Protect Home Office
  • Acronis True Image 2021
  • Acronis True Image 2020
  • Acronis True Image 2019
  • Acronis True Image 2018
  • Acronis True Image 2017
  • Acronis True Image 2016

Operating Systems: 

  • macOS

Last update: 30-09-2021

Acronis Cyber Protect Home Office
Acronis Cyber Protect Home Office supports High Sierra

Acronis True Image 2021
Acronis True Image 2021 supports High Sierra.

Acronis True Image 2020
Acronis True Image 2020 supports High Sierra.

Acronis True Image 2019
Acronis True Image 2019 supports High Sierra.

Acronis True Image 2018
Acronis True Image 2018 supports High Sierra starting with build 9620. You can download the latest build in your Acronis account (if you need assistance downloading the build, see this article).

Acronis True Image 2017
Acronis True Image 2017 supports Sierra (10.12), but does not support High Sierra (10.13). Disk-level backup and bootable media creation might fail. If you experience these issues, please contact Acronis Customer Central.
Backing up files and folders works in High Sierra.

Acronis True Image 2016, 2015
Does not support High Sierra. You can purchase an upgrade to the latest version of the product.
